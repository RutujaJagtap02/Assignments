Wrapper Class

A wrapper class in Java is a class that converts a primitive data type into an object.
Each primitive type (like int, char, boolean, etc.) has a corresponding wrapper class (like Integer, Character, Boolean, etc.) in the java.lang package.

We Need Wrapper class Because Java has two types of data:
Primitive types: like int, char, boolean â€” these are not objects.
Objects: needed when working with things like collections (ArrayList, HashMap, etc.).
But Java collections can only store objects, not primitive types. So we use wrapper classes to convert primitives into objects.

Examples of Wrapper Classes

Primitive Type Wrapper Class
int Integer
char Character
boolean Boolean
double Double

Ex:
int num = 5; // primitive
Integer boxedNum = num; // wrapped into an object
int unboxed = boxedNum; // unwrapped back to primitive

This Wrapper Class Allows primitives to be used in collections (like ArrayList, which only store objects)

Generics

Generics allow us to write classes, interfaces, and methods that can work with different data types
Instead of writing separate classes or methods for each data type, we can define them once with a placeholder (like T) and Then the compiler enforce type safety at compile time.
Then we can caught Errors before running the program.
The compiler checks that only the correct type is used
