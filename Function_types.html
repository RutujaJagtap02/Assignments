Function Types

1]Anonymous Function

An anonymous function means a function without a name.

this used as arguments to other functions

anonymous function Cannot be reused unless stored in a variable

        EX:
            const show = function () {
             console.log("Anonymous function stored in a variable");
            };
            show();

2]Arrow Function

Arrow functions are a shorter way to write functions in JavaScript.

Uses => instead of function
Arrow Function are good for short, simple functions 

            Ex:
                const add = (a, b) => a + b;
                console.log(add(2, 3)); 
                // Output will be 5 //

3]Callback Function

A callback is a function passed as an argument to another function.
this function Used when we want one function to run after another
this is used Commonly in asynchronous operations (like loading data) 

    Ex:
        function greet(name, callback) {
             console.log("Hi " + name);
                callback();
        }

        greet("Bob", function () {
            console.log("This is the callback function");
        });

Introduction to DOM

The DOM (Document Object Model) in JavaScript is like a map of our webpage that JavaScript uses to find and change things on the page.

What is DOM

DOM stands for Document Object Model.
It’s a tree-like structure that represents all the elements on a webpage (like headings, paragraphs, buttons, etc.).
When a webpage loads, the browser creates this structure so JavaScript can interact with it.

What can JavaScript do with the DOM?
Change text or images on the page Add or remove elements (like showing a
popup or hiding a menu) Change styles (like making text bold or changing
colors) Respond to user actions (like clicking a button or typing in a form)

Imagine our webpage is a TV. The DOM is the remote control that JavaScript uses to change the channel, adjust the volume, or turn the TV on/off.

Events in JavaScript

    Events are how JavaScript knows something has happened on a webpage—like a user clicking a button or typing in a form. JavaScript can then respond to these events.
    Common Event Types:
         Mouse Events: click, dblclick, mouseover, mouseout
         Keyboard Events: keydown, keyup, keypress
         Form Events: submit, change, focus, blur
         Window Events: load, resize, scroll

What Is Camel Casing?

        Camel casing (or camelCase) is a naming style used in JavaScript where:
        The first word is lowercase
        Each following word starts with a capital letter
        No spaces or underscores
